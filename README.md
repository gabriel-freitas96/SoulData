# 🔐 SoulData - Herança Digital

O **SoulData** é uma aplicação web que permite aos usuários armazenar e proteger informações de ativos digitais (como contas, criptomoedas e documentos), garantindo que possam ser acessadas por herdeiros em caso de inatividade.

---

## 🚀 Funcionalidades

* 🔑 Sistema de **Login e Registro**
* 📦 Cadastro de **ativos digitais**
* 🧠 Organização em um **cofre digital**
* 👤 Definição de **herdeiro**
* ⏳ Configuração de **tempo de inatividade**
* 🔐 Simulação de **criptografia de dados**
* 🔒 Proteção do cofre com senha
* 💎 Sistema de **planos (Básico e Premium)**

---

## 🖥️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Estrutura SPA (Single Page Application)

## 🔐 Segurança (Importante)

Atualmente o projeto utiliza:

* Simulação de criptografia (Base64)
* Armazenamento em memória (não persistente)

⚠️ **Não é seguro para uso real em produção.**

### Melhorias futuras:

* Implementar backend (Node.js ou Java)
* Criptografia real (AES / SHA-256)
* Autenticação com JWT
